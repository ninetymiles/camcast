# Camera Streaming Cast Application for Android

An simple app to cast camera live streaming to RTMP or SRT server.

## Features

> [StreamPack](https://github.com/ThibaultBee/StreamPack)

- [x] RTMP live streaming
- [x] SRT live streaming
- [x] Zoom on pinch
- [x] Focus on tap
